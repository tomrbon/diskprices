## Introduction

External hard drives seem simple—just pick a capacity and buy, right? But the choices are overwhelming: portable vs desktop, HDD vs SSD, USB-A vs USB-C, different speeds, different reliability, different use cases.

This guide cuts through the confusion. You'll learn exactly which external drive suits your needs, what features are worth paying for, and which specifications you can safely ignore.

### Portable External Drives

Small, bus-powered drives you can slip in a pocket:

**Characteristics:**
- 2.5-inch HDD or SSD inside
- Powered by USB (no external power)
- 1-5TB typical capacity for HDD
- 500GB-4TB typical for SSD
- Slower than desktop drives

**Best for:**
- Laptop backup
- File transfer between computers
- Travel storage
- Expanding laptop storage

**Popular models:**
| Drive | Type | Capacity | Price | Speed |
|-------|------|----------|-------|-------|
| WD Elements Portable | HDD | 5TB | $120 | 100-120 MB/s |
| Seagate Backup Plus Slim | HDD | 2TB | $65 | 100-120 MB/s |
| Samsung T7 Shield | SSD | 1TB | $110 | 1000 MB/s |
| SanDisk Extreme | SSD | 1TB | $100 | 1050 MB/s |

### Desktop External Drives

Larger drives that require external power:

**Characteristics:**
- 3.5-inch HDD inside
- Requires separate power adapter
- 4-22TB capacities
- Faster than portable HDDs
- Stay on desk, not portable

**Best for:**
- Desktop backup
- NAS alternative (single drive)
- Media storage
- Large file archives

**Popular models:**
| Drive | Capacity | Price | Speed |
|-------|----------|-------|-------|
| WD Elements Desktop | 8TB | $150 | 150-180 MB/s |
| Seagate Expansion Desktop | 10TB | $170 | 150-180 MB/s |
| WD My Book | 14TB | $250 | 180-200 MB/s |

### External SSDs

Fast, durable solid-state drives:

**Characteristics:**
- No moving parts
- 5-20x faster than HDDs
- More durable (survive drops)
- Smaller and lighter
- Higher cost per TB

**Best for:**
- Video editing on external drive
- Gaming (load games directly)
- Boot drives
- Frequent travel
- Photo backup in the field

## HDD vs SSD: Which Should You Choose?

### Speed Comparison

| Drive Type | Read Speed | Write Speed | Load Time (10GB file) |
|------------|------------|-------------|------------------------|
| Portable HDD | 100-120 MB/s | 80-100 MB/s | 90 seconds |
| Desktop HDD | 150-200 MB/s | 120-180 MB/s | 55 seconds |
| SATA SSD | 500-550 MB/s | 450-500 MB/s | 18 seconds |
| NVMe SSD | 1000-2000 MB/s | 1000-2000 MB/s | 5-10 seconds |

### Durability Comparison

| Scenario | HDD | SSD |
|----------|-----|-----|
| Drop from desk | May fail | Usually fine |
| While operating | High risk | No risk |
| Temperature | Sensitive | Tolerant |
| Vibration | Problematic | No issue |
| Long-term storage | 3-5 years | 5-10 years |

### Cost per Terabyte

| Drive Type | $/TB | Notes |
|------------|------|-------|
| Portable HDD | $25-30 | Budget option |
| Desktop HDD | $15-20 | Best value per TB |
| External SSD | $60-100 | Premium, faster |

### Use Case Recommendations

| Use Case | Recommendation | Why |
|----------|----------------|-----|
| Backup only | Desktop HDD | Cheap, large capacity |
| Laptop backup | Portable HDD or SSD | Portable |
| Video editing | External SSD | Speed matters |
| Gaming | External SSD | Load times |
| Photo storage | Portable HDD or SSD | Depends on frequency |
| Cold storage | Desktop HDD | Best $/TB |
| Travel | External SSD | Durability |

## Interface and Connectivity

### USB Standards

| USB Version | Max Speed | Real-World SSD Speed |
|-------------|-----------|---------------------|
| USB 3.0 / 3.1 Gen 1 | 5 Gbps | 400-450 MB/s |
| USB 3.1 Gen 2 | 10 Gbps | 800-1000 MB/s |
| USB 3.2 Gen 2x2 | 20 Gbps | 1500-2000 MB/s |
| Thunderbolt 3/4 | 40 Gbps | 2500-3000 MB/s |

### Which USB Do You Need?

**For HDDs:**
- USB 3.0 is sufficient (HDDs can't exceed 200 MB/s)
- USB-C or USB-A doesn't matter for speed

**For SSDs:**
- USB 3.1 Gen 2 for portable SSDs (1000 MB/s)
- Thunderbolt for high-end NVMe enclosures (2000+ MB/s)

**Cable Considerations:**
- USB-C to USB-C: Most modern computers
- USB-C to USB-A: Older computers
- Adapters work but may limit speed

### Thunderbolt: Worth the Premium?

Thunderbolt external SSDs cost 2-3x more than USB alternatives:

| Drive | Interface | Speed | Price (1TB) |
|-------|-----------|-------|-------------|
| Samsung T7 | USB 3.2 Gen 2 | 1000 MB/s | $110 |
| Samsung X5 | Thunderbolt 3 | 2800 MB/s | $200 |
| OWC Envoy Express | Thunderbolt 3 | 2600 MB/s | $180 |

**You need Thunderbolt if:**
- Editing 4K/8K video directly from the drive
- Running virtual machines from external
- Large file transfers daily (saving minutes per transfer)

**USB 3.1 is enough if:**
- General backup
- Photo storage
- Gaming
- Document storage

## Capacity Planning

### How Much Storage Do You Need?

| User Type | Typical Need | Recommendation |
|-----------|--------------|----------------|
| Casual user (documents, photos) | 500GB-1TB | 2TB drive |
| Photographer | 2-4TB | 4TB drive |
| Video editor | 4-8TB | 8TB+ or multiple drives |
| Gamer | 1-2TB per 10 games | 2-4TB SSD |
| Backup drive | 2-3x system size | 2x largest drive |

### The 50% Rule

For backup drives, buy at least 2x your data size:
- Current data: 2TB
- Buy: 4TB drive
- Allows: Growth + version history + space for recovery

### Future-Proofing

Consider 3-year growth:
- Average user: Data grows 20-30% per year
- Photographer: 50-100% per year
- Video creator: 100%+ per year

## Features That Matter (and Don't)

### Features Worth Paying For

**Hardware Encryption:**
- Protects data if drive is stolen
- Look for AES-256 hardware encryption
- WD My Passport, Seagate Backup Plus have this

**Drop Protection:**
- SSDs naturally durable, but some are reinforced
- Samsung T7 Shield: 3m drop resistance
- LaCie Rugged: Designed for field work

**Water/Dust Resistance:**
- IP ratings (IP65, IP67) indicate protection
- LaCie Rugged SSD Pro: IP67
- Important for field photographers

**Backup Software:**
- WD SmartWare, Seagate Toolkit
- Can be convenient but not essential
- Time Machine and File History work with any drive

### Features You Can Ignore

**Manufacturer Backup Software:**
- Often clunky and limited
- Use OS-native tools instead

**"Cloud Backup" Included:**
- Usually just a free trial
- You'll pay eventually

**RGB Lighting:**
- Looks cool, adds cost, no function

**Proprietary Connectors:**
- Avoid drives with non-standard cables
- If the cable breaks, you need a specific replacement

## Shucking External Drives

Some external drives contain high-quality internal drives:

### Why Shuck?

- WD My Book often contains WD Red or White drives
- Seagate Expansion sometimes has IronWolf
- Cheaper than buying internal drive

### Risks

- Void warranty
- May damage case
- Some drives have proprietary connectors

### Popular Drives to Shuck

| External | Possible Internal | Value |
|----------|-------------------|-------|
| WD My Book 8TB+ | WD Red/White | Great |
| WD Elements Desktop 10TB+ | WD White | Good |
| Seagate Expansion 8TB+ | Seagate Barracuda | Okay |

### The 3.3V Pin Issue

Some WD drives use a 3.3V pin that prevents boot in desktop PCs:

**Fix:** Tape over pin 3 on the SATA power connector, or use a power supply that doesn't provide 3.3V.

## Conclusion

Choosing an external hard drive doesn't have to be complicated:

**For backup:** Desktop HDD for capacity, portable for convenience
**For speed:** External SSD with USB 3.1 Gen 2
**For durability:** SSD with drop protection
**For value:** Desktop HDD has best $/TB

Buy for your use case, not for specs you don't need. A $100 portable HDD backs up documents just as well as a $200 SSD—save the premium for when speed matters.
